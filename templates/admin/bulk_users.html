Instructions
Required CSV Columns:
  • username - Unique username (e.g., john.doe)
  • password - Initial password
  • first_name - User's first name
  • last_name - User's last name
  • email - Valid email address
Optional CSV Columns:
  • phone - Phone number
  • department - Department name
  • title - Job title
  • posix_enabled - yes/no for POSIX account
  • login_shell - Shell for POSIX users (default: /bin/bash)
  • home_directory - Home dir for POSIX users (default: /home/username)
Important Notes:
  • CSV file must have headers matching the column names above
  • Usernames must be unique - existing users will be skipped
  • Passwords should meet your organization's security requirements
  • POSIX users get system access with UID/GID assignment
  • Standard users get basic LDAP authentication only
